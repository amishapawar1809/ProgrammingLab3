World Wide Web terms:

Internet
- It is a global network.Unlike online services, which are centrally controlled, the Internet is decentralized by design. Each Internet computer, called a host, is independent. Its operators can choose which Internet services to use and which local services to make available to the global Internet community. There are a variety of ways to access the Internet. For most of its existence the Internet was primarily a research and academic network. Today people and businesses around the world can use the Internet to retrieve information, communicate and conduct business globally, and access a vast array of services and resources online.
Web browser
- It is a software which is used to browse and display pagea available over internet. It sends request to server for web based documents and services. It has no processing model. It stores user data in cookies in local machine.
Web server
- It is a software which provides documents when requested by web browsers. It sees and approves requests made by web browsers and sends document in response. It has three processing models. It provides an area to store the website. It can be installed anywhere, but it needs to be on network.
Hyperlink
- It is a reference to data that the user can follow by clicking or tapping. It points to whole document or to a specific element within a document.
Hypertext
- It is text displayed on a computer display or other electronic devices with references i.e.hyperlinks to other text that reader can immediately access. It also sometimes used to describe tables, images and other presentational content formats with integrated hyperlinks.
HTTP
- Hypertext Transfer Protocol is an application protocol for distributed, collaborative, hypermedia information systems. It is the foundation of data communication for the WWW. HTTP resources are identified and located on the network by URL.
